# Healthcare Analytics Dashboard

## Project Overview

Developed an interactive 3-page Healthcare Analytics Dashboard using Power BI to analyze patient admissions, demographics, and revenue trends.

## Tools Used

* Power BI
* DAX
* Power Query
* Excel/CSV Dataset

## Dashboard Pages

### 1. Executive Summary

* Total Patients
* Total Revenue
* Average Billing Amount
* Average Stay Days
* Admission Trends
* Lab Test Results Analysis

### 2. Patient Demographics

* Gender Distribution
* Age Group Analysis
* Blood Type Distribution
* Medical Condition Analysis
* Insurance Provider Analysis

### 3. Revenue Analytics

* Revenue by Admission Type
* Revenue by Medical Condition
* Revenue by Insurance Provider
* Top Hospitals by Revenue
* Revenue Trend Analysis

## Key Features

* Interactive slicers
* Page navigation buttons
* Dynamic filtering
* DAX measures and KPIs
* Multi-page dashboard design

## Skills Demonstrated

Power BI, DAX, Data Visualization, Data Cleaning, Dashboard Design, Business Analytics
